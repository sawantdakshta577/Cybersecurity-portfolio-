# Event ID 4624 Analysis

## Alert Name

Successful Account Logon

## Event ID

4624

## Description

Event ID 4624 is generated when a user, service, or system account successfully logs on to a Windows system.

## Environment

* Computer Name: SOC-LAB-01
* Operating System: Windows 11
* Log Source: Windows Security Log

## Investigation

The event was observed in the Windows Security Log.

Observed Details:

* Event ID: 4624
* Logon Type: 5 (Service Logon)
* Account Name: SYSTEM
* Account Domain: NT AUTHORITY
* Authentication Package: Negotiate
* Elevated Token: Yes

## Analysis

Logon Type 5 indicates a service logon. This type of event is commonly generated when Windows services start or authenticate.

The login was performed by the SYSTEM account using the legitimate Windows process services.exe. No remote IP address or suspicious authentication activity was identified.

## Findings

The activity is consistent with normal Windows operating system behavior and was generated by a trusted system service.

## Impact

No security impact detected.

## Verdict

False Positive / Benign Activity

## Analyst Conclusion

The successful logon was determined to be legitimate system activity. No signs of unauthorized access, privilege misuse, or malicious behavior were observed.
